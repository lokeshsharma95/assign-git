1•What is a gitignore file and what should it include for a Java project?
Ans : .gitignore stores all the extensions of files which is ignored by git and is not tracked by git.
These can be some specific files which should not be stored in a remote repository. It can also contain pattern to ignore
all the files matching the pattern. Ex. *.log
